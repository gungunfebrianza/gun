Navigation Bar Sizes

Add the ggf-small, ggf-large, ggf-xlarge or ggf-xxlarge to change the size of the navigation (will change the font-size of each link).

Default:

  • Home
  • Link 1
  • Link 2
  • Link 3

ggf-large:

  • Home
  • Link 1
  • Link 2
  • Link 3

ggf-xlarge:

  • Home
  • Link 1
  • Link 2
  • Link 3

ggf-xxlarge:

  • Home
  • Link 1
  • Link 2
  • Link 3

If you want more space between each link, instead of increasing the font-size, you could add the ggf-padding-classes to each link:

  • Home
  • Link 1
  • Link 2
  • Link 3
